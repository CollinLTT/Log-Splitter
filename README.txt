TO RUN PROGRAM:
1. Open the LogSplitter application file and run as administrator.
2. Fill out the appropriate information.
3. Press "Split File".
4. Once pressed the process will begin and inform you when finished.

!IMPORTANT!
You MUST fill out all of the input information before pressing "Split File" or an error message will appear.

Each line will be split upon reading the "Search String" input.
