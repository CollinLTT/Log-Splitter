TO RUN PROGRAM:
1. Open the LogSplitter.exe file and run it as administrator.
2. Fill out the appropriate information.
3. Press "Split File".
4. Once pressed the splitting process will begin and inform you when finished.
--------------------------------------------------------------------------------------------------------------
!IMPORTANT INFO!
1. You MUST fill out all input information before pressing "Split File" or an error message will appear.

2. Each line will be split upon reading the "Search String" input.

3. There is a default file split size of 1,400,000 Excel lines.
